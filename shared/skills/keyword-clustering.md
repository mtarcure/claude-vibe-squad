# Keyword Clustering

Group candidate keywords by search intent and topic, and map each cluster to a page.

## Steps
1. Gather candidate keywords from grounded search (not invented volumes).
2. Tag each keyword's search intent: informational / navigational / transactional / commercial.
3. Cluster by intent + topic so one cluster expresses one user need.
4. Map each cluster to a single target page/content piece.
5. Flag cannibalization risk where two pages would compete for the same cluster.

## Acceptance
- Every keyword is tagged with an intent.
- Each cluster maps to exactly one page (no two pages target the same cluster).
- Volumes/keywords are grounded, not fabricated.
