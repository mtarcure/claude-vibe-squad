# Narrative Structure

Build story arc, quest graph, and dialogue outline that serve play, within an existing game-design contract.

## Steps
1. Define the arc and stakes; state the player fantasy the story reinforces.
2. Build a character/quest graph with prerequisites and branches; give each node a stable ID.
3. Align narrative beats to level pacing so story serves the moment-to-moment loop.
4. Outline dialogue and key choices; note where a branch changes mechanics.
5. Flag sensitive/regulated themes (age rating, real persons/events) for content review before ship.

## Acceptance
- The quest graph has no dangling prerequisites; every branch resolves.
- Beats map to levels; story serves play (call out where it doesn't).
- Sensitive/regulated themes are flagged, not silently shipped.
